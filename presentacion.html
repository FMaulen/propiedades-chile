Contexto

¿Cuánto vale una casa en Santiago?

El mercado inmobiliario de la Región Metropolitana es enorme y opaco. Los precios varían drásticamente entre comunas, y no existe una herramienta accesible que le diga a una persona cuánto debería pagar —o cobrar— por una propiedad.

Construimos un sistema que recopila datos reales, los limpia e integra con información socioeconómica, entrena modelos predictivos, y los pone a disposición a través de una API y un dashboard interactivo.

¿Por qué estas tecnologías?

  • SQLite — Base de datos liviana, sin servidor, ideal para el volumen de datos que manejamos
  • FastAPI — Genera documentación Swagger automática y es el estándar actual en Python
  • Random Forest — Captura relaciones no lineales entre precio y características de la propiedad
  • Pipeline de sklearn — Encapsula preprocesamiento y modelo en un solo archivo .pkl
